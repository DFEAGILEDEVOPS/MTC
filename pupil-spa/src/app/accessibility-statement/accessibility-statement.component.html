Accessibility statement

We're constantly working to make the multiplication tables check (MTC) accessible and user-friendly for all pupils, regardless of physical, cognitive or emotional impairment.

You can read a blog about accessibility on GOV.UK.

Our commitment to accessibility

The Standards and Testing Agency is committed to making websites accessible, in accordance with The Public Sector Bodies (Websites and Mobile Applications) Accessibility Regulations 2018.

This accessibility statement applies to the MTC:

  • pupil site
  • teacher site

Conformance status

We seek to ensure that all aspects of the service meet the Web Content Accessibility Guidelines (WCAG) 2.1 at Level AA. The MTC is partially conformant with WCAG 2.1 level AA.

Partially conformant means that some parts of the content do not fully conform to the accessibility standard.

Pupil site

The MTC does not support third-party screen readers due to compatibility issues with the timed element of the check.

To address this, MTC provides a number of access arrangements that can be applied to the pupil check. The following table shows which access arrangements are compatible with each device.

Access arrangement Recommended browser/device
Audible time alert Not compatible with IE11
Colour contrast Any browser/device
Font size Any browser/device
Input assistance Any browser/device
'Next' button between questions Any browser/device
Question reader Not recommended for IOS 9.3
Not compatible with IE11
Remove on-screen number pad Any browser/device (External keyboard required for tablets)

Teacher site

MTC does not currently have any built-in access arrangements for the teacher site.

The MTC teacher site is being designed to be compatible with recent versions of the following screen readers:

  • job access with speech (JAWS)
  • non-visual desktop access (NVDA)
  • voiceOver

The site is also being designed to be compatible with:

  • basic operating system screen magnifiers, such as ZoomText magnifier
  • speech recognition software, such as Dragon Naturally Speaking
  • operating system speech packages

Get help for Teacher Site

AbilityNet provides guidance about how to:

  • make your mouse easier to use
  • use your keyboard instead of a mouse
  • talk to your device
  • make your device talk to you
  • make text larger
  • change your colours
  • magnify the screen
