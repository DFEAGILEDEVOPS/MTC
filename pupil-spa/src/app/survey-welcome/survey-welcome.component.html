Multiplication tables check ICT survey

The purpose of this survey is to help us understand about information and communications technology (ICT) used in schools.

Complete this survey using your school ICT. If you have a range of devices, please run this survey on them one at a time.

Following the connection test, please give us feedback on ICT issues you may have come across.

You will be able to preview the multiplication tables check system after the connections test.

This test will determine your:

  • browser settings
  • connection speed
  • device type (if available)
  • operating system
  • processing speed
  • screen width

During this process, data files will be downloaded, held temporarily and then deleted. All data will be anonymised unless contact details are provided in your feedback.
